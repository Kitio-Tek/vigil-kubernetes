# Support

Athos Kubernetes is an open-source project maintained by Kitio-Tek and the
community. The project is in preview and does not yet ship a paid support
channel.

## Channels

- GitHub Issues: file bugs, feature requests, questions.
- GitHub Discussions: design conversations and broader questions.
- Security: see [SECURITY.md](SECURITY.md) for the disclosure process.

## SLA

Best-effort. Pull requests with a clear motivation and reproducible test case
get prioritised. Security reports are acknowledged within five working days.
