# Supported Versions

While Athos Kubernetes is in 0.x preview, only the most recent minor release
receives security updates. Patch releases land on `release-MAJOR.MINOR`
branches.

| Version | Supported |
|---|---|
| 0.9.x | Yes |
| 0.8.x | Security fixes only |
| 0.7.x | No |
| 0.6.x | No |
| 0.5.x | No |
| 0.4.x | No |
| 0.3.x | No |
| 0.2.x | No |
| 0.1.x | No |

## Supported Kubernetes versions

| Kubernetes | Supported |
|---|---|
| 1.30 | Yes |
| 1.29 | Yes |
| 1.28 | Best-effort |
| < 1.27 | No |

## Supported PostgreSQL major versions

| PostgreSQL | Supported |

| 15 | Yes |
| 14 | Best-effort |
